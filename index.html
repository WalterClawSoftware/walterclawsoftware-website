Transcript workspace

ClipScript

macOS + Windows

ClipScript is the right fit when you want to fetch transcripts from YouTube videos with captions, keep a reusable local archive, and turn transcript text into notes, scripts, captions, and repurposed content.

  • Paste a YouTube URL or video ID and fetch the transcript directly.
  • Search across saved transcripts and reopen previous work.
  • Export clean working copy for writing and production workflows.
$9.99 per desktop version No sign-in required App Store + Microsoft Store live

Transcript cleanup

Transcript Rescue

macOS + Windows

Transcript Rescue is the right fit when the whole job is cleaning rough transcript text into readable output fast, without needing a larger transcript workspace.

  • Paste raw transcript text or open common transcript files.
  • Use Clean or Polish mode with optional timestamps and digest summaries.
  • Save readable output and cleanup reports without subscription baggage.
$9.99 per desktop version Cleanup-focused workflow Apple App Store under review
View Transcript Rescue Watch demo
Bug handoff cleanup

Repro Pack

macOS + Windows

Repro Pack is the right fit when a messy customer report, support thread, or copied issue note needs to become a developer-ready repro packet before it reaches engineering.

  • Separate reported facts from assumptions, missing context, and follow-up questions.
  • Build general, Jira-ready, Linear-ready, and GitHub-ready drafts.
  • Keep the workflow small, clear, and desktop-friendly.
$9.99 per desktop version Known vs unknown stays visible Planned after Transcript Rescue review
View Repro Pack See packet sample
YouTube banner safety

BannerSafe for YouTube

macOS + Windows

BannerSafe is the right fit when you already have channel banner art and need to know whether the logo, mascot, subtitle, or other protected content survives YouTube's device crops.

  • Import an existing PNG or JPG instead of starting from a blank editor.
  • Preview the full YouTube canvas, desktop strip, and mobile/logo safe region.
  • Export a YouTube-ready 2560 x 1440 image after placement checks.
Version 0.1.1 $9.99 per desktop version Preview trust first

App icon creation

App Icon Creator

macOS + Windows

App Icon Creator is the right fit when a 1024 x 1024 logo PNG still looks too small in the Dock, taskbar, installer, or storefront. It creates standardized Mac and Windows icon files with proof before release.

  • Open or drag in a logo PNG and inspect visible artwork bounds.
  • Generate standardized PNG, macOS ICNS, Windows ICO, and proof sheets.
  • Use readiness evidence before updating installers, apps, or storefront media.
